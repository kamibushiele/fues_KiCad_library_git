# これまで作ってきたKiCadライブラリ
## Kicadバージョン
__7.0.0__

## 構成

``` txt
    +-library-(コンポーネントライブラリ)
    |   +-*.lib
    |   +-*.dcm
    |
    +-modules
        +-3Dmodels
        |   +-*(3Dモデルフォルダ)
        |   |   +-*.wrl
        |   |
        |   +-...
        |
        +-*.pretty(フットプリント)
        +-...
```

## 導入
### まずは
適当な場所にクローンしてくる

### 環境変数の追加

| 名前        | パス                         |
| ----------- | ---------------------------- |
| `KIFUESMOD` | `[リポジトリのパス]/modules` |
| `KIFUESLIB` | `[リポジトリのパス]/library` |

### コンポーネントライブラリの追加

各ライブラリを選択して追加する
